# SDKWork Browser Config Profiles

Source-controlled browser engine and deployment profile templates.

| Path | Purpose |
| --- | --- |
| `profiles/browser.default.yaml` | Default browser engine profile |
| `profiles/browser.cef.yaml` | CEF engine profile |
| `profiles/browser.servo.yaml` | Servo engine profile |

Runtime secrets and per-environment overrides belong in ignored local paths or `~/.sdkwork/sdkwork-browser`.
